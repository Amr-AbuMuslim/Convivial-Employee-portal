# 👨‍💼 Convivial Employee Portal

A modern **Employee Management Portal** built with **Vite**, **React.js**, and **TypeScript**.  
This portal allows admins and users to manage employees, teams, authentication, and more —  
featuring a beautiful UI, reusable components, and secure API integration.

---

## 🚀 Tech Stack

| Category | Technologies |
|-----------|---------------|
| Frontend | React.js (TypeScript), Vite |
| UI | Tailwind CSS, Framer Motion, Lucide Icons |
| State Management | React Hooks, Context API |
| API | Axios |
| Routing | React Router DOM |
| Form Handling | Controlled Components |
| Notifications | Custom Toast Component |

---

## ⚙️ Features

✅ User Authentication (Create Password, Login, Logout)  
✅ Role-based Access (Admin / User)  
✅ Employee Management (Add, Edit, Delete, Pagination)  
✅ Team & Department Management  
✅ Responsive UI with Animations  
✅ Custom Toast Alerts  
✅ Secure API Integration

## 🛠️ Installation & Setup

Follow these steps to run the project locally 👇

### 1️⃣ Clone the repository

git clone https://github.com/Amr-AbuMuslim/Convivial-Employee-portal.git
cd Convivial-Employee-portal

2️⃣ Install dependencies
npm install

3️⃣ Run the development server
npm run dev


The app will be available at http://localhost:5173

🏗️ Build for Production
npm run build


The production build will be generated inside the dist/ folder.

To preview the build locally:

npm run preview

You can deploy the project to:
